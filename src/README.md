# Código-fonte

- `layer7d` — daemon principal (futuro)
- `classifier` — integração nDPI / classificação
- `policy` — motor de políticas
- `events` — modelo e emissão de eventos
- `runtime` — estado, reload, integração PF/DNS
- `common` — utilitários compartilhados

PoC nDPI: pasta **`poc_ndpi/`** + `scripts/build/build-poc-freebsd.sh` · doc resultados em `docs/poc/`.
